Task 5: Arrays & Basic Data Analysis Program

Description:
This is a Java console application developed as part of
Java Developer Internship Task 5.

The program demonstrates the use of arrays and basic
data analysis operations.

Features:
- Stores numerical data using 1D array
- Calculates Sum, Average, Maximum and Minimum
- Performs manual sorting without using inbuilt methods
- Uses 2D array to display matrix data
- Handles ArrayIndexOutOfBoundsException
- Displays formatted output in console

Concepts Used:
- 1D and 2D Arrays
- Loops (for loop)
- Conditional statements
- Manual sorting logic
- Exception handling
- Console output formatting

Tools Used:
- Java Programming Language
- IntelliJ IDEA / Eclipse
- Windows OS
- GitHub

How to Run:
1. Open the project in IntelliJ IDEA or Eclipse
2. Compile the Java file
3. Run the program
4. View the output in the console/Terminal

Output:
The program displays:
- Sum of array elements
- Average value
- Maximum and Minimum values
- Sorted array
- 2D array elements
- Exception handling message
